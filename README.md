# 🎵 Spotify Clone

A simple and responsive **Spotify-like music player** built using **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Features

- 🎧 Play/Pause songs
- ⏭️ Next/Previous track control
- 📻 Interactive UI with animations
- 🖼️ Album cover and background image
- 🎼 Local audio files for playback
- 🔁 Responsive design for all devices

---

## 📁 Project Structure

```
Spotify Clone/
├── covers/          # Album cover images
├── songs/           # Audio files (e.g. .mp3)
├── index.html       # Main HTML structure
├── style.css        # Page styling
├── script.js        # Music player functionality
├── logo.png         # Site logo
├── bg.jpg           # Background image
├── playing.gif      # Playing animation
└── README.md        # Project info (this file)
```

## 🛠️ How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/spotify-clone.git
   ```

2. Open the project folder and launch `index.html` in your browser.

3. Make sure your browser allows local media access (for songs to play properly).
